To Do Hero

Be a hero and set your goals

To Do Hero is my version of a to do app. You can save your objectives by making an account. The web app was made with Meteor and the DB that I use is MongoDB.

You can see the code here. To make it work download Meteor, access the root of the folder with command prompt and then write: meteor. It will run on your local host.
